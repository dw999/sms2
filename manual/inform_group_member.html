3.10 Group admin - Inform group member
Under emergency situation, group administrator and system administrator can enforce to send out notification email(s) / Telegram message(s) to urge off-line users to check their SMS messages.

Select 'Inform Member' on group menu (figure 3.3.1), then screen like figure 3.10.1 will be displayed. In there, a default message 'You have important message, please check for it ASAP." is put on the message input area, you may accept it by simply click on the 'Send' button, or you can input your desired notification message on it, and then click the 'Send' button. However, please remember that DON'T put any secret information on the notification message.

Note:
This function is for common message groups only, not for private message groups.
