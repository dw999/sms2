6.3 Edit feature
To modify a feature, click on the feature icon or program file name, screen similar to figure 6.3.1 will be displayed. In there, you can upload a new icon image to replace the original icon, upload an updated version of feature program, modify related database schema by providing a SQL script file, active or deactivate feature on decoy web site, or change active feature sorting order.

Icon file and program file will be updated if you give new file on the corresponding input file, and database schema will be changed if you provide a SQL script file. After you input all data you wanted to change, press the 'Save' button to update.
 

Figure 6.3.1
 
Remark:
If any new javascript library files and/or image file(s) is/are required by your updated feature, please put javascript library files and image files to directories '/www/pdatools/js' and '/www/pdatools/images' respectively. This function cannot help you to deploy javascript library and image files.
