6.1 Feature list
If connection mode of your SMS server is 0 or 3, SMS users will be switched to decoy site after login. In the first web page of the decoy site, you will find some icons on it (similar to figure 6.1.1). Every icon on this page represent a feature. If you are system administrator, you have right to add new feature, modify existing feature, select which features to be shown or hidden on the first page of the decoy site.
 

Figure 6.1.1
 
To access the feature list, click on the 'Setup' button on the left top corner to bring in the menu (figure 6.1.2), and select the item 'Feature Setup'. Then feature list will be displayed (figure 6.1.3).
 

Figure 6.1.2
 

Figure 6.1.3
 
As you may have already noted on the feature list (figure 6.1.3), item with yellow background colour is shown on the first page of the decoy site, but item with gray background colour does not. It means that you could have many features on your decoy site, but just let some of them be used. And you could change the displaying order of shown features also. I will show you how to do it in next section.
