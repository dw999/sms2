4.4 Lock/Unlock user
If you are system administrator, you have right to lock and unlock any user of any ranking. i.e. Include other system administrators

To lock an user, select item 'Lock/Unlock user' on main menu (figure 2.1.3), then screen like figure 4.4.1 will be seen, which let you desire to lock or unlock an user. Select radio button 'Lock User', then click the 'Next' button. Figure 4.4.2 will be shown to let you select active user(s) to lock. Select the check-box in front of desired user's alias, and then click 'Save' button to lock selected user(s).

Locked user will be forced logout from the system immediately if he/she is still online, and he/she cannot login to SMS again. His/Her messages will be kept on the system, and also his/her message group profiles. i.e. He/she won't be kicked out from message group(s) he/she belongs to. However, all his/her messages will show that he/she is an inactived user as his/her alias colour is changed from red to grey.

Similarly, if you want to unlock a locked user, select 'Unlock User' radio button on figure 4.4.1, and select desired locked user(s) on figure 4.4.2, and click 'Save' button to unlock user(s).

Please note that only appropriate users will be listed on figure 4.4.2. Therefore, if it has no any appropriate user available, the user list will be empty.
