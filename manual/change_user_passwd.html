4.5 User password
This function is used to change happy password of all active users, and mainly supported SMS server in connect mode 1. However, it can be also served SMS server in other connection modes.

First you must find out those user(s) you need to change his/her/their happy password. Just select item 'User Password' on main menu (figure 2.1.2), screen like figure 4.5.1 will be shown. You may input full or partial user name/user alias/email to filter out specified user(s), then click the "Next Page" button. Note: As the warning message in this screen shown, if you leave the user filtering input field blank, it may cause system unresponse or timeout in next step, if the active user list is too long.
 

Figure 4.5.1
 
If your specified user(s) exist, his/her/their record(s) will be displayed in this page like figure 4.5.2 shown. Otherwise, a message will show you that no user is found.
 

Figure 4.5.2
 
Just click on any cell of a row of specified user, figure 4.5.3 will be displayed. In there you could input a new happy password for this user, and click "Update Password" to save the new happy password.
